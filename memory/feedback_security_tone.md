---
name: Tono de avisos de seguridad
description: Bajar la intensidad de las advertencias cuando el contexto es de bajo riesgo (cuentas dedicadas o throwaway)
type: feedback
---

**Regla:** Para cuentas dedicadas solo a mi uso (throwaway/bot), no acumular múltiples advertencias de seguridad ("borra el mensaje", "rota la contraseña", "activa 2FA"). Una nota breve basta — o ninguna si Ruben ya calibró el riesgo.

**Why:** Ruben dijo "Solo la vas a usar tú, no hay peligro" tras yo apilarle 3 avisos sobre una clave de email. Le sobra el tono paranoico cuando él ya decidió el nivel de riesgo.

**How to apply:** Para cuentas reales/personales → recomendar buenas prácticas. Para cuentas dedicadas a mi uso: guardar la credencial con chmod 600, confirmar en una línea, seguir adelante. Sin sermón.
