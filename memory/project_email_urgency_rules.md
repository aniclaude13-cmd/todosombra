**Criterios de urgencia** (cualquiera de estas condiciones):

**Dominios siempre urgentes:**
- `@cmbarcelo.com`, `@awma.es`, `@tecnitoldo.es`, `@espacius.com`

**Keywords en asunto o cuerpo:** urgente, asap, importante, hoy, vence

**Contenido urgente (cualquier remitente):**
- Solicitud de presupuesto
- Problema activo: queja, incidencia, fallo, error, retraso, reclamación, avería
- Tema legal/administrativo con plazo inminente

En caso de duda → INFO (no urgente).

**Formato de notificación:**
```
🕐 [HH:MM del correo]
👤 [Remitente]
📌 [Asunto]
📝 [Resumen 2-3 líneas]
```

**Why:** Criterios base indicados el 2026-05-11, ampliados el 2026-05-15 con @tecnitoldo.es, @espacius.com, solicitudes de presupuesto y el formato actual.
